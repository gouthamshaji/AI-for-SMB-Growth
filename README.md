# AI-for-SMB-Growth
An AI-powered marketing co-pilot designed for home-based D2C sellers with a simple, chat-first interface to create and schedule content.

**Dukaan Saathi** is an AI-powered marketing assistant tailored specifically for home-based D2C sellers. Designed with a familiar, WhatsApp-first chat interface, it empowers busy entrepreneurs to easily generate daily content, preview posts, schedule campaigns, and track simple business outcomes without the complexity of traditional marketing tools.
